## Overview

The SEO Fundamentals skill provides principles for improving search engine visibility through Google's E-E-A-T framework, Core Web Vitals optimization, technical SEO best practices, and content quality guidelines.

## What This Skill Provides

- **E-E-A-T framework**: Experience, Expertise, Authoritativeness, Trustworthiness signals
- **Core Web Vitals**: Performance metrics that impact rankings (LCP, INP, CLS)
- **Technical SEO**: Site structure, sitemaps, robots.txt, canonical tags, HTTPS
- **Content SEO**: Page elements (title tags, meta descriptions, heading hierarchy)
- **Schema markup types**: Article, Organization, Person, FAQPage, Product, Review
- **AI content guidelines**: How to use AI-generated content while following E-E-A-T
- **Ranking factors**: Prioritized list from content quality to technical fundamentals
- **Measurement tools**: Search Console, Analytics, PageSpeed Insights

## Use Cases

- Improving organic search rankings and visibility
- Optimizing content for search engines while maintaining quality
- Implementing technical SEO best practices
- Creating schema markup for rich search results
- Balancing AI-assisted content creation with E-E-A-T requirements
- Measuring and tracking SEO performance

## Key Principles

### E-E-A-T Framework

- **Experience**: First-hand knowledge and real examples
- **Expertise**: Credentials and depth of knowledge
- **Authoritativeness**: Backlinks, mentions, industry recognition
- **Trustworthiness**: HTTPS, transparency, accurate information

### Core Web Vitals as Ranking Factors

- **LCP**: < 2.5s (loading performance)
- **INP**: < 200ms (interactivity)
- **CLS**: < 0.1 (visual stability)

### Content Quality Over Quantity

Comprehensive, unique, and regularly updated content outperforms thin or duplicate content.

## Ranking Factors Priority

1. Quality, relevant content
2. Backlinks from authority sites
3. Page experience (Core Web Vitals)
4. Mobile optimization
5. Technical SEO fundamentals

## AI Content Guidelines

| Do | Don't |
|----|---------|
| AI draft + human edit | Publish raw AI content |
| Add original insights | Copy without value |
| Expert review | Skip fact-checking |
| Follow E-E-A-T | Keyword stuffing |
