  <Steps>
    <Step title="Install the CLI">
      Install Antigravity Kit globally or run it directly with npx:
      
      ```bash npm
      npm install -g @vudovn/ag-kit
      ```
      
      ```bash npx
      npx @vudovn/ag-kit init
      ```
      
      This installs the `.agent` folder containing all templates into your project.
    </Step>
    
    <Step title="Configure your editor">
      Antigravity Kit works with AI-powered editors like **Cursor** and **Windsurf**.
      
      <Note>
        To ensure slash commands appear in the chat suggestion dropdown, add `.agent/` to your local exclude file (`.git/info/exclude`) instead of `.gitignore`.
      </Note>
    </Step>
    
    <Step title="Start coding with AI agents">
      Just describe what you need—agents are automatically selected based on your request:
      
      ```
      You: "Add JWT authentication"
      AI: 🤖 Applying @security-auditor + @backend-specialist...
      ```
      
      Or use slash commands for complex workflows:
      
      ```
      /create landing page with hero section
      /debug why login fails
      /test authentication flow
      ```
    </Step>
  </Steps>
